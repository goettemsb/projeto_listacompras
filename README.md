# Lista de Compras 🛒

Um aplicativo Flutter simples e prático para gerenciar sua lista de compras!

---

## 🚀 Funcionalidades

- Adicionar produtos à lista
- Remover produtos da lista
- Visual moderno e responsivo
- Interface intuitiva
